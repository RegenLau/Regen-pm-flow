# Affinity Diagram (KJ Method)

Cluster unstructured qualitative data — quotes, observations, ideas — into natural groupings to reveal themes and patterns.

## Steps

1. **Capture data points** — Write each observation, quote, or idea on its own note (one insight per note).
2. **Spread and read** — Lay all notes out so the full dataset is visible.
3. **Group silently** — Move notes into clusters that feel related; let groupings emerge naturally rather than forcing categories.
4. **Name each cluster** — Write a short header that captures the theme of each group.
5. **Identify meta-themes** — Look across clusters for higher-order patterns, hierarchies, or relationships between groups.

## Output Format

A hierarchical list of theme groups, each with a descriptive header, the individual data points underneath, and a summary of cross-cutting patterns and key insights.
