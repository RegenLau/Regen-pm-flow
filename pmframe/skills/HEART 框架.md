# HEART Framework (Google)

Measure user experience across five dimensions with Goals, Signals, and Metrics for each.

## Steps

1. **Happiness** — define metrics for user satisfaction (e.g., NPS, CSAT, ease-of-use ratings).
2. **Engagement** — measure depth and frequency of interaction (e.g., session length, actions per session).
3. **Adoption** — track new user acquisition and feature uptake (e.g., new signups, feature activation rate).
4. **Retention** — measure return usage over time (e.g., D7/D30 retention, churn rate).
5. **Task Success** — assess efficiency and effectiveness (e.g., completion rate, time-on-task, error rate).

## Output Format

HEART table with one row per dimension, columns for Goal, Signal, and Metric, plus recommended measurement cadence and tooling.
