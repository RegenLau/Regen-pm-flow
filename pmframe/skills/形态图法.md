# Morphological Chart (Zwicky)

Decompose a problem into independent dimensions, enumerate possible values for each, and systematically explore combinations.

## Steps

1. Define the problem clearly and identify its independent **dimensions** (parameters, functions, or sub-problems).
2. For each dimension, list all feasible **values** (solutions, options, or approaches) without filtering.
3. Build a matrix with dimensions as rows and values as columns.
4. Systematically combine one value from each dimension to generate candidate solutions — flag promising and novel combinations.
5. Evaluate top combinations against feasibility, desirability, and viability criteria.

## Output Format

Morphological matrix table, list of generated combinations (highlighting non-obvious ones), and evaluation of top 3-5 candidates with selection rationale.
