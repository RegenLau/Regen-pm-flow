# PESTLE Analysis

Systematically scan six macro-environmental dimensions for external forces that create opportunities or threats.

## Steps

1. **Political** — government policy, trade regulations, political stability, tax policy.
2. **Economic** — growth rates, inflation, exchange rates, disposable income trends.
3. **Social** — demographics, cultural trends, lifestyle changes, education levels.
4. **Technological** — emerging tech, R&D activity, automation, digital infrastructure.
5. **Legal** — employment law, consumer protection, data privacy, IP regulations. **Environmental** — climate, sustainability mandates, resource scarcity, waste regulations.

## Output Format

PESTLE table with key factors per dimension, impact assessment (positive/negative, high/medium/low), time horizon, and recommended strategic responses.
