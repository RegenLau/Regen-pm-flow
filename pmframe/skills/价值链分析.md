# Value Chain Analysis (Porter)

Decompose the organization into strategically relevant activities to find where value is created or lost.

## Steps

1. Map **Primary Activities**: Inbound Logistics, Operations, Outbound Logistics, Marketing & Sales, Service.
2. Map **Support Activities**: Firm Infrastructure, HR Management, Technology Development, Procurement.
3. For each activity, assess cost drivers, value contribution, and current performance level.
4. Identify linkages — where one activity's performance affects another's cost or effectiveness.
5. Pinpoint activities that are sources of differentiation or cost advantage, and flag underperforming areas.

## Output Format

Value chain diagram with all activities labeled, cost and value assessment per activity, linkage analysis, and strategic recommendations for strengthening competitive advantage.
