# Business Model Canvas (Osterwalder)

Visualize an entire business model on a single page using nine interdependent blocks.

## Steps

1. Define **Customer Segments** — who are you creating value for?
2. Articulate the **Value Proposition** for each segment.
3. Map **Channels** (awareness, purchase, delivery) and **Customer Relationships** (self-service, dedicated, community).
4. Identify **Revenue Streams** (per segment) and **Key Resources/Activities/Partners** needed to deliver.
5. Estimate **Cost Structure** and validate that revenues exceed costs sustainably.

## Output Format

A completed 9-block canvas table with concise entries per block, plus a brief viability summary noting the riskiest assumptions.
