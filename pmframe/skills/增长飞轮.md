# Growth Flywheel

Map the virtuous cycles where each step in the loop accelerates the next, creating compounding growth.

## Steps

1. Identify the **core action** users take that creates value
2. Map what that action produces as an **output** (content, data, network, etc.)
3. Trace how the output **attracts** more users or increases engagement
4. Identify what makes returning users **do more** of the core action
5. Draw the full loop and label the **force multiplier** at each connection
6. Find friction points that slow the flywheel and design interventions

## Output Format

**Flywheel Loop:**
[Action] -> [Output] -> [Attraction] -> [More Users] -> [Action] (loop)

**Force Multipliers:**
- [Connection 1]: [What amplifies it]
- [Connection 2]: [What amplifies it]

**Friction Points:** [Where the loop breaks down]
**Acceleration Plan:** [Top interventions to spin faster]
