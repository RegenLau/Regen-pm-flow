# Threshold of Viable Ideas

Define the minimum bar an idea must clear before it deserves time, money, or attention.

## Steps

1. Define threshold criteria: market size, feasibility, strategic fit, differentiation potential, and time-to-value.
2. Set minimum acceptable levels for each criterion (binary pass/fail or minimum score).
3. Evaluate each idea against every criterion — document evidence, not opinions.
4. Reject ideas that fail any critical threshold; flag borderline ideas for further research.
5. Advance passing ideas into deeper evaluation (business case, prototyping, or experimentation).

## Output Format

Criteria checklist with thresholds, evaluation table scoring each idea, clear pass/fail/borderline verdict per idea, and next steps for passing ideas.
