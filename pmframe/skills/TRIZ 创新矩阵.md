# TRIZ

Resolve design contradictions using a structured innovation methodology. Use when improving one product parameter worsens another, and you need a creative breakthrough rather than a compromise.

## Steps

1. Define the **problem** as a contradiction: improving Parameter A worsens Parameter B.
2. Map both parameters to the 39 standard TRIZ engineering parameters.
3. Look up the **contradiction matrix** to find suggested inventive principles.
4. Review each suggested principle from the 40 inventive principles list.
5. Generate concrete solution concepts by applying each principle to your context.
6. Evaluate and select the most promising concept.

## Output Format

A contradiction statement, the mapped TRIZ parameters, the suggested inventive principles with brief descriptions, and 2-3 concrete solution concepts with feasibility notes.
