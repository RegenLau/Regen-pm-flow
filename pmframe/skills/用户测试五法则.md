# 5-User Usability Test (Nielsen)

Uncover approximately 85% of usability issues by testing with just 5 representative users. Use when you need fast, actionable UX feedback on a prototype or live product.

## Steps

1. **Define goals** — pick 3-5 key tasks users should be able to complete.
2. **Recruit** — find 5 participants matching your target persona.
3. **Prepare** — write a test script with task scenarios (not instructions) and set up screen recording.
4. **Facilitate** — observe each participant attempting the tasks; use think-aloud protocol; do not help or lead.
5. **Analyze** — compile issues by frequency and severity; group into themes.
6. **Report** — prioritize fixes by severity x frequency and share with the team.

## Output Format

A usability report with: task completion rates, a severity-ranked issue list with screenshots, observed user quotes, and a prioritized list of recommended fixes.
