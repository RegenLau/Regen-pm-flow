# Elevator Pitch Template

Structure a compelling one-paragraph pitch that communicates value clearly and memorably.

## Steps

1. Define the **target customer** — "For [specific audience]..."
2. State the **need or problem** — "...who [struggle with / need to]..."
3. Name the **product and category** — "[Product] is a [category]..."
4. Articulate the **key benefit** — "...that [primary value delivered]."
5. Differentiate — "Unlike [alternative], we [unique differentiator]."

## Output Format

A single pitch paragraph following the template: "For [target], who [need], [product] is a [category] that [benefit]. Unlike [competitor], we [differentiator]." Plus brief notes on supporting proof points.
