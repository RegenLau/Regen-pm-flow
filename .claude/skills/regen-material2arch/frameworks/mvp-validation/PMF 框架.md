# Product-Market Fit Assessment

Measure PMF using the Sean Ellis test and supporting signals to determine if you should scale or iterate.

## Steps

1. Run the **Sean Ellis survey**: "How would you feel if you could no longer use this product?" (Very disappointed / Somewhat disappointed / Not disappointed)
2. Calculate the **PMF score** -- percentage answering "Very disappointed" (target >= 40%)
3. Analyze **retention curves** -- do they flatten (PMF) or trend to zero (no PMF)?
4. Check **organic growth signals** -- word of mouth, inbound interest, NPS
5. Segment responses to find which user cohort has the strongest fit
6. If below 40%, identify what "very disappointed" users love and double down

## Output Format

- **Sean Ellis Score:** [X]% very disappointed (threshold: 40%)
- **Verdict:** [Has PMF / Approaching PMF / No PMF]
- **Strongest Segment:** [User cohort with highest score]
- **What they love:** [Core value proposition]
- **Retention curve shape:** [Flattening / Declining]
- **Next action:** [Scale / Iterate on X / Pivot]
