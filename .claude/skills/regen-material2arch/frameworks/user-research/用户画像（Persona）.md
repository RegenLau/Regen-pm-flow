# Persona Creation

Construct realistic user archetypes grounded in research data to align teams on who they are building for.

## Steps

1. Gather data — user interviews, surveys, analytics, support logs, sales call notes.
2. Identify behavioral patterns and cluster users into distinct segments.
3. For each persona, define: name, demographics, role, goals, frustrations, behaviors, tools used, and a representative quote.
4. Add context: a day-in-the-life scenario showing when and why they encounter your product.
5. Validate personas against real user data; retire any that lack evidence.

## Output Format

Persona card per archetype with photo placeholder, demographics, goals, frustrations, key behaviors, scenario narrative, and data sources used.
