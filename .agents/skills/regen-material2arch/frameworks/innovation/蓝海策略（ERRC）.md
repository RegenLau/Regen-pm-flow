# Blue Ocean ERRC Grid

Break free from competitive benchmarking by systematically rethinking which factors to Eliminate, Reduce, Raise, and Create.

## Steps

1. List the **factors the industry competes on** (features, price, service, etc.)
2. **Eliminate** -- which factors can be completely removed without losing core value?
3. **Reduce** -- which factors can be reduced well below the industry standard?
4. **Raise** -- which factors should be raised well above the industry standard?
5. **Create** -- which factors should be introduced that the industry has never offered?
6. Draft the new **value curve** and compare it to incumbents

## Output Format

| Eliminate | Reduce | Raise | Create |
|-----------|--------|-------|--------|
| [Factor] | [Factor] | [Factor] | [Factor] |
| [Factor] | [Factor] | [Factor] | [Factor] |

**New value proposition:** [One-sentence summary of differentiated offering]
