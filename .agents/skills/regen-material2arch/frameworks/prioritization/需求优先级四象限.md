# Priority Quadrant

Classify tasks by urgency and importance to focus effort on what truly matters.

## Steps

1. List all tasks, requests, or items requiring prioritization.
2. Classify each into one of four quadrants: **Q1** (Urgent + Important) — do immediately; **Q2** (Important + Not Urgent) — schedule and protect time; **Q3** (Urgent + Not Important) — delegate or batch; **Q4** (Neither) — eliminate.
3. For Q1 items, assign owners and deadlines.
4. For Q2 items, block dedicated time — this is where strategic work lives.
5. Actively reduce Q3/Q4 work through delegation, automation, or removal.

## Output Format

2x2 quadrant grid with tasks placed in each cell, action plan per quadrant, and a recommended weekly time allocation split.
